# BMI計算機_s1131547

## 功能介紹
* **計算BMI**：輸入身高(cm)與體重(kg)，自動計算 BMI 值（四捨五入至小數點後兩位）。
* **體位判定與視覺提示**：判斷體位（體重過輕、健康體位、體位過重、輕度肥胖、中度肥胖、重度肥胖），並搭配文字顏色顯示。
* **格外功能**:
  
  **1. 歷史紀錄清單**：自動記錄每一次的測量時間與結果，方便追蹤與比較。
  
  **2. 一鍵清除功能**：清空輸入框與狀態，並自動將游標移回身高輸入框，提升使用者操作的順暢度。

  **3. 視覺提示變更為莫蘭迪色系**: 我將體位判定的文字視覺提示改成莫蘭迪色系，讓原本鮮艷的顏色變為更柔和的顏色，讓版面更和諧好看。
